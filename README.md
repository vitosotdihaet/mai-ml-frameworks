# Лабораторные работы по "Прикладным системам и фреймворкам ИИ"
Репозиторий содержит результаты выполнения лабораторных работ по изучению и сравнению различных алгоритмов машинного обучения.

## Содержание

### Выполненные работы:
1. **Лабораторная работа №1** - Исследование алгоритма KNN
2. **Лабораторная работа №2** - Исследование логистической и линейной регрессии  
3. **Лабораторная работа №3** - Исследование решающего дерева
4. **Лабораторная работа №4** - Исследование случайного леса

## Цель исследований
Каждая лабораторная работа следует единому паттерну:
1. Изучение распределений и зависимостей для задач классификации и регрессии
2. Создание и оценка бейзлайна с использованием sklearn
3. Улучшение бейзлайна через эксперименты с препроцессингом и гиперпараметрами
4. Самостоятельная имплементация алгоритма
5. Сравнение результатов кастомной и библиотечной реализаций

Был использован [датасет выживаемости лошади](https://www.kaggle.com/datasets/yasserh/horsesurvivalprognostication) по ее характеристикам при болезни для классфикации исхода лечения.

Был использован [датасет зарплат сотрудников Cassandra](https://www.kaggle.com/datasets/rockyt07/cassandra-employee-dataset) по информации о сотруднике для предсказывании его зарплаты.

## Используемые метрики

### Классификация:
- Accuracy, Precision, Recall, F1-score

### Регрессия:
- MAE (Mean Absolute Error)
- MSE (Mean Squared Error) 
- R² (коэффициент детерминации)

## Выводы
В процессе выполнения работ было исследовано влияние различных техник препроцессинга и настройки гиперпараметров на качество моделей. Были реализованы модели машинного обучения на Python
